# CleanManager
🧺 CleanManager - Sistema de Gestión Logística con Supabase & React📖 Resumen del ProyectoDryClean Flow es una plataforma web de alto rendimiento diseñada para la digitalización de tintorerías locales. El sistema centraliza la recepción de prendas, el control de estados de limpieza y la comunicación automatizada con el cliente.El ProblemaLas tintorerías tradicionales sufren de saturación de stock (prendas que el cliente olvida recoger) y pérdida de tiempo productivo en gestión telefónica manual.La SoluciónUna arquitectura reactiva que utiliza Supabase Realtime para el seguimiento de prendas y Edge Functions para disparar notificaciones de WhatsApp en el momento exacto en que una prenda está lista, optimizando así el flujo de salida y el espacio físico del local.🛠️ Stack TecnológicoTecnologíaRol en el ProyectoReact + ViteFramework de interfaz de usuario con renderizado eficiente.TypeScriptTipado estático para reducir errores en tiempo de ejecución.SupabaseBackend-as-a-Service (BaaS) que incluye Auth, DB y Storage.PostgreSQLBase de datos relacional para la gestión compleja de pedidos.Tailwind CSSDiseño de interfaz moderno, responsive y optimizado para operarios.Lucide ReactSet de iconos vectoriales para una navegación intuitiva.⚙️ Arquitectura de Datos y AutomatizaciónEl sistema utiliza la potencia de PostgreSQL Triggers y Supabase Edge Functions para la lógica de negocio:Inserción/Update: El operario marca una prenda como LISTA_PARA_ENTREGA.Trigger: Supabase detecta el cambio en la tabla pedidos.Webhook/Function: Se invoca una función que conecta con la API de mensajería (Twilio/WPP).Notificación: El cliente recibe un mensaje con el código de recogida.
